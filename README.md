# Tokoflix

## Installation

For setup this mini project, you need Node v8.10.0, NPM 3.5.2, webpack 4,and Linux Environtment.

from root folder :

**BASH**

```bash
$ cd /{YOUR_PATH_FOLDER}/
git clone "https://github.com/adreansyah/tokoflix.git"
$ cd tokoflix/
```

**NPM**

```bash
npm install
npm start
```
